Code Example

Kata - Array Deep Count: Your task is to create a function deepCount that returns the number of ALL elements within an array, including any within inner-level arrays. The input will always be an array. 
                                    
function deepCount(arr) {
    let countSum = arr.length;
    for (let item of arr)
        if (Array.isArray(item)) countSum += deepCount(item);
    return countSum;
}
